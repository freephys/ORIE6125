ORIE6125 Project Proposal

List of members: (Michael) Chek Hin Choi, Yingjie (Tom) Fei

Project content:

In this project, we aim at implementing one (or more if time permits) statistics and machine learning algorithms using C++ and Python.

Here, we list the algorithms that we consider implementing:

  1. Particle filtering
  2. Principal Component Analysis (PCA)
  3. Kalman filter

Our primary focus will be on particle filtering methods. If time permits, we will apply the above algorithms to some real data set and analyze the results.
